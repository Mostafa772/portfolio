Neural Network From Scratch

Tech: Python · NumPy · Date: Jan 2024

This project is a minimal, well-documented implementation of a feed-forward neural network using only NumPy. It was designed as a learning resource to understand backpropagation and optimization without the abstraction of higher-level frameworks.

Highlights

  • Forward & backward passes implemented from first principles
  • Supports multiple activation functions and batch gradient descent
  • Clean notebook demonstrating experiments and results
